Dreamweaver

The projects shown here are from the only web design class taught as part of Sinclair's Visual Communications Associate Degree. Most students entering this class have no prior knowledge of web design, so HTML structure and CSS are taught from the ground up.

After becoming more familiar with the proper structure of a web page, students are able (but not required) to move into Adobe Dreamweaver, which offers some tools for more easily developing the code for their web site. More advanced topics such as CSS animation, JavaScript, and responsive design are discussed but are not required skills for the course.

For this project, students gathered "10 Best Websites" from across the Internet to showcase in a gallery. Each site was analyzed based on a number of factors, including content, usability, and overall design. Additionally, the students were required to include a list of provided events, a "featured designer page", and a list of helpful web design resources. On the Home page, students included a slideshow, an advertisement, and portions of content from their Resources page. All of the pages included a navigation menu and a functional footer.

Use the images below allow to view the finished web sites. Please note that the sites are NOT optimized for mobile devices and are generally fixed-width designs.
