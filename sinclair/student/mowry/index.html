Search Engine Optimization

SEO is the process of maximizing the number of visitors to a particular website by ensuring that the site appears high on the list of results returned by a search engine. "The key to getting more traffic lies integrating content with search engine optimization and social media marketing".


Designing a Website

When designing a website you need to do some brainstorming and out ideas to paper. From there draw ideas for a web page layout. There are a lot more aspects that go into designing the final layout of the web page.


Hosting a Website

Basically Web Hosting refers to a place where all website owners store their websites. Your essential files, from documents to videos to images will all be kept in a web host. Web hosting services are categorized into plans. When you start to hunt for a web hosting plan, its easy to be overwhelmed by the long list of available options.

Cascading Style Sheets

CSS is a language for specifying how your documents are presented to users and how they are styled, lais out, ect. A document is usually a text file structured using a markup language called HTML. From that document you use CSS to visually style your structured HTML.


Usability Considerations

Considering how people are going to use your page will help you figure out where navigation, layout and color scheme come into play. Creating a usuable and simple layout can make or break your page.
