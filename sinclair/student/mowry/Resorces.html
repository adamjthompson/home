Usuability Considerations

Usuability (aka User Friendly) is different approaches to what made a product 'usable' splinteresd between looking at it with the view of the product (Ergonomic Design); looking at it from the user and user performance, which involves how easy the product is to use, if it's to be used in the real world. These are some links to help you better understand how to make your website 'Usable'

Hosting Websites

When you make a website and want other people to see it, you will need to publish (or upload) it with a web hosting service. Web hosting services work by storing your website files in high-powered computers (webservers) connected to a very fast network.
