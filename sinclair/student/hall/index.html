search engine optimization

Search engine optimization (SEO) is the process of maximizing the number of visitors to a particular website by ensuring that the site appears high on the list of results returned by a search engine. "The key to getting more traffic lies in integrating content with search engine optimization and social media marketing." SEO is important to any business that operates online, and many don't realize it needs to be built into the web design process -- not added in later. Learn more.

usability considerations

Usability is a measure of a user’s ability to arrive on a site, use it easily and complete the desired task. When designing a website there is a flow. You can’t only focus on page design, assuming everything will flow later. “Usability” refers to the ease of access and/or use of a product or website. It’s a sub-discipline of user experience design. Although user experience design (UX Design) and usability were once used interchangeably, we must now understand that usability provides an important contribution to UX; however, it’s not the whole of the experience. We can accurately measure usability. Learn more.

hosting a website

Web hosting is the activity or business of providing storage space and access for websites. Websites are hosted or stored on servers. Once you register your first domain name, it is time to find a place to host your first site. This is your web hosting. Basically, web hosting is the computer that is attached to the Internet where you will place the files for your website. Your domain name, once all the setup is complete, will then refer anyone who connects using their browser to the computer where your files are stored. Learn more.

cascading style sheets

Cascading style sheets (CSS) describe how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. Web browsers apply CSS rules to a document to affect how they are displayed. Learn more.

designing a website

Web design is the process of creating websites. It encompasses several different aspects, including webpage layout, content production, and graphic design. While the terms web design and web development are often used interchangeably, web design is technically a subset of the broader category of web development. Learn more.
